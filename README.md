# Reporter

A C program to retrieve information about your Linux machine

------------------------------------------------------------------------

# How to Install

Open the terminal and type following commands

* `apt update`

* `apt install git`

* `git clone https://github.com/UncleJ1ck/Reporter`

* `cd Reporter`

* `make`

* `sudo ./reporter`
